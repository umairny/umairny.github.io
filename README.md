# Umair's Portfolio

A premium, interactive personal portfolio website designed to showcase a unique blend of Graphic Design and Full Stack Web Development skills. Built with modern web technologies, this project emphasizes visual excellence, smooth animations, and a seamless user experience.

## ✨ Key Features

- **Premium Aesthetic**: Features a sleek, modern design with glassmorphism effects, vibrant color palettes, and a "dark mode first" approach.
- **Interactive UI**: Powered by **Framer Motion** for smooth page transitions, element animations, and scroll-linked effects.
- **3D Gallery**: specific showcase section (`/gallery`) featuring 3D project cards with depth effects and filtering capabilities (Development vs. Design).
- **Dynamic Content**:
  - **Bento Grid Layout**: A stylish "About Me" section utilizing a grid layout to present bio, journey, and philosophy.
  - **Tech Arsenal**: An interactive skills section with filtering tabs for Development and Design tools.
- **Theme Support**: Includes a robust dark/light theme toggler (defaulting to a premium dark theme).
- **Responsive Design**: Fully responsive layout ensuring a consistent experience across all devices.

## 🛠️ Tech Stack

- **Core**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS3 (Custom Properties / Variables) for maximum control and performance.
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

   The application should now be running at local address (usually `http://localhost:5173`).
